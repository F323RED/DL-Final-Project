/*********** READ ME ***********/

本程式可接受二個參數，分別是 "原始 State Table 位置" 及 "縮減後 State Table 位置"。
若無任何參數輸入，則讀取預設檔案 "FSM_table.txt" 及 "Reducted_FSM_table.txt"。

範例 : 
cmd > "DL Final Project.exe"	aaa.txt	bbb.txt

Custom file : aaa.txt, bbb.txt
Reading file...

Number of table : 1
Table 1
Input size : 1
Output size : 1
Number of states : 7
0 a a 0
1 a b 0
~
~
~
End of table 1
